PostgreSQL cluster-level monitoring dashboards, including:

- [PGSQL Cluster](pgsql-cluster): Main dashboard for a PGSQL cluster
- [PGRDS Cluster](pgrds-cluster): RDS version of PGSQL Cluster, focusing on PostgreSQL-native metrics
- [PGSQL Activity](pgsql-activity): Session/load/QPS/TPS/locks for PGSQL cluster
- [PGSQL Replication](pgsql-replication): Replication, slots, and pub/sub for PGSQL cluster
- [PGSQL Service](pgsql-service): Service, proxy, routing, and load balancing for PGSQL cluster
- [PGSQL Databases](pgsql-databases): Database CRUD, slow queries, and table statistics across all instances
- [PGSQL Patroni](pgsql-patroni): HA status and Patroni component status for cluster
- [PGSQL PITR](pgsql-pitr): PITR context for point-in-time recovery assistance
