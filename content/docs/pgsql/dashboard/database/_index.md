PostgreSQL database-level monitoring dashboards, including:

- [PGSQL Database](pgsql-database): Main dashboard for a single PGSQL database
- [PGCAT Database](pgcat-database): Database info retrieved directly from database catalog
- [PGSQL Tables](pgsql-tables): Table/index access metrics within a single database
- [PGSQL Table](pgsql-table): Detailed info for a single table (QPS/RT/index/sequence...)
- [PGCAT Table](pgcat-table): Detailed table info from database catalog
- [PGSQL Query](pgsql-query): Detailed info for a query type (QPS/RT)
- [PGCAT Query](pgcat-query): Query details from database catalog
- [PGCAT Locks](pgcat-locks): Activity and lock wait info from database catalog
- [PGCAT Schema](pgcat-schema): Schema info from database catalog
