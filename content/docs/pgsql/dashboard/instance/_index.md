PostgreSQL instance-level monitoring dashboards, including:

- [PGSQL Instance](pgsql-instance): Main dashboard for a single PGSQL instance
- [PGRDS Instance](pgrds-instance): RDS version of PGSQL Instance, focusing on PostgreSQL-native metrics
- [PGCAT Instance](pgcat-instance): Instance info retrieved directly from database catalog
- [PGSQL Persist](pgsql-persist): Persistence metrics: WAL, XID, checkpoint, archive, IO
- [PGSQL Proxy](pgsql-proxy): Detailed metrics for a single HAProxy load balancer
- [PGSQL Pgbouncer](pgsql-pgbouncer): Metrics overview for a single Pgbouncer connection pooler
- [PGSQL Session](pgsql-session): Session and active/idle time metrics for a single instance
- [PGSQL Xacts](pgsql-xacts): Transaction, lock, TPS/QPS related metrics
- [PGSQL Exporter](pgsql-exporter): Self-monitoring metrics for Postgres and Pgbouncer exporters
